# Samaira Cloud Intelligence

Samaira Cloud Intelligence is a multi-tenant FinOps platform designed to provide deep visibility into cloud resource utilization, cost optimization, and automated remediation.

Built with FastAPI, PostgreSQL, Airflow, and React, Samaira enables secure AWS account onboarding via cross-account IAM roles, encrypted credential storage, and tenant-isolated analytics.

The platform aims to deliver intelligent cost forecasting, anomaly detection, and SLO-aware cost insights for modern cloud-native environments.
